# hands-on-astronomer

Hands-on Astronomer - Learning Airflow

## 01. Install Atro CLI

```shell
brew install astro
```

## 02. Create project and initialize Airflow with Astro

```shell
mkdir -p ~/astro && cd ~/astro && astro dev init
```

## 03. Start Airflow with Astro

```shell
astro dev start
```

## 04. Stop Airflow with Astro

```shell
astro dev stop
```
